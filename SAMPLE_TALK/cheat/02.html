• You can add notes for each slides in the same manner you add images
  • The naming convention is 01.html - 20.html
  • Theoretically you can style your notes however you like using HTML and inline CSS
  • But there is no need to use HTMl at all, if you just want plain, unformatted text
  • For this example I used Headline (h1), paragraph (p)
  • and a simple list (ul and li tags, no custom styling)

Some Notes for Event Organizers

  • Best practice is to set up the whole event on your web server
  • Ask speakers to send you their slides and notes as properly formatted files
  • Upload everything to your event directory (name it whatever you like)
  • Name the sub directories for the talks 01, 02, 03, etc., depending on how many speakers you have
  • Add and edit the _config.jason files in all talk directories
